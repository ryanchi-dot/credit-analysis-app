# 快速开始指南

本文档帮助你快速启动和使用银行信贷分析助手网页版。

## 5分钟快速启动

### 步骤1：安装依赖

```bash
cd web_app
pip install -r requirements.txt
```

### 步骤2：配置环境变量

创建 `.env` 文件：

```bash
cp .env.example .env
```

编辑 `.env` 文件，设置你的智能体API地址：

```bash
AGENT_API_URL=https://your-agent-url/run
```

### 步骤3：运行应用

```bash
streamlit run app.py
```

### 步骤4：访问应用

浏览器打开：http://localhost:8501

### 步骤5：注册账号

1. 点击"注册"标签
2. 输入用户名和密码（密码至少6位）
3. 点击"注册"按钮

### 步骤6：登录

1. 点击"登录"标签
2. 输入用户名和密码
3. 点击"登录"按钮

### 步骤7：开始使用

1. 在输入框中输入企业名称（如：腾讯控股有限公司）
2. 选择分析重点（可选）
3. 上传参考资料（可选）
4. 点击"开始分析"按钮
5. 等待智能体生成报告（5-8分钟）
6. 下载报告

## 常见问题

### Q1: 忘记密码怎么办？

A: 目前不支持密码找回，可以重新注册一个账号。

### Q2: 如何上传文件？

A: 点击文件上传器，选择文件（PDF、Word、Excel、图片），然后点击"开始分析"。

### Q3: 报告在哪里下载？

A: 分析完成后，会显示4个报告链接，点击链接即可下载。

### Q4: 如何切换用户？

A: 点击侧边栏的"退出登录"按钮，然后重新登录。

### Q5: 数据会丢失吗？

A:
- 本地部署：数据保存在本地，不会丢失
- Render免费版：应用休眠后数据会保留，但文件系统会清空

## 数据安全说明

### 用户数据隔离

- 每个用户的数据完全独立
- 用户只能访问自己的数据
- 文件按用户ID分别存储

### 密码安全

- 密码使用SHA-256加密存储
- 管理员无法查看明文密码

### 文件安全

- 文件存储在 `data/uploads/user_{user_id}/` 目录
- 用户只能访问自己的文件

## 🚀 快速开始

### 步骤1：部署Agent（如果还未部署）

将Agent部署到Coze平台或自己的服务器，获取API端点URL。

### 步骤2：在Coze平台创建插件

1. 登录Coze平台
2. 进入"插件市场" → "创建插件"
3. 选择"自定义API插件"
4. 填写基本信息：
   - 名称：银行信贷分析助手
   - 描述：基于企业公开信息生成专业的授信分析报告
   - 图标：📊

5. 配置API：
   - 端点URL：替换 `manifest.json` 中的 `https://your-agent-url/stream_run` 为你的实际URL
   - 请求方法：POST
   - 请求头：`Content-Type: application/json`

6. 定义输入参数（参考 `manifest.json`）

7. 定义输出参数（参考 `manifest.json`）

8. 测试插件

9. 发布插件

### 步骤3：在其他Bot中使用插件

在Bot的对话流程中，添加"调用插件"节点：
```
节点类型：调用插件
插件名称：银行信贷分析助手
参数：
  - company_name: {{input_company_name}}
  - analysis_focus: {{input_analysis_focus}}
```

## 💡 常见使用场景

### 场景1：单个企业分析
```
用户输入：分析腾讯的授信情况
→ 调用插件
→ 返回4个部分的下载链接
```

### 场景2：批量分析多个企业
```
工作流：批量授信分析
→ 循环调用插件
→ 收集所有报告链接
→ 返回汇总结果
```

### 场景3：集成到信贷审批流程
```
工作流：信贷审批流程
→ 用户提交企业信息
→ 调用插件生成授信分析报告
→ 人工审核
→ 审批决策
```

## ⚠️ 注意事项

1. **API端点**：确保 `manifest.json` 中的URL已替换为实际的部署地址
2. **网络连接**：确保Coze平台能够访问你的Agent服务
3. **超时设置**：报告生成需要5-8分钟，注意设置合理的超时时间
4. **错误处理**：建议添加错误处理和重试机制

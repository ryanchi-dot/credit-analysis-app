# 快速启动指南

## 🚀 3 步快速启动小程序

### 第1步：准备环境（5分钟）

#### 1.1 安装 Python
- 下载：https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"
- 验证安装：打开命令行，输入 `python --version`

#### 1.2 安装微信开发者工具
- 下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 安装后使用微信扫码登录

#### 1.3 注册小程序账号
- 访问：https://mp.weixin.qq.com/
- 注册小程序账号（需要 300 元认证费，或使用个人免费账号）

---

### 第2步：启动后端服务（10分钟）

#### 2.1 安装依赖
```bash
# 进入后端目录
cd backend_miniprogram

# 安装依赖包
pip install -r requirements.txt
```

#### 2.2 修改代码中的路径
打开 `backend_miniprogram/agent_wrapper.py`，修改第 10 行：

```python
# 原代码
project_root = os.path.abspath(os.path.join(os.path.dirname(__file__), '..'))

# 修改为（根据实际情况调整）
project_root = os.path.abspath(os.path.dirname(__file__))
```

#### 2.3 启动服务
```bash
python main.py
```

看到以下输出表示成功：
```
============================================================
银行信贷分析助手 API 服务
============================================================
服务地址: http://127.0.0.1:8000
API 文档: http://127.0.0.1:8000/docs
============================================================
```

**重要**：保持这个命令行窗口打开，不要关闭！

---

### 第3步：运行小程序（5分钟）

#### 3.1 创建小程序项目
1. 打开微信开发者工具
2. 点击 "+" 新建项目
3. 填写信息：
   - 项目名称：银行信贷分析助手
   - 目录：选择 `miniprogram_example` 目录
   - AppID：使用你的小程序 AppID（或选择测试号）
4. 点击"新建"

#### 3.2 修改后端地址
打开 `miniprogram_example/app.js`，修改第 6 行：

```javascript
// 如果是本地测试，保持不变
apiBaseUrl: 'http://localhost:8000'

// 如果部署到云服务器，修改为实际地址
// apiBaseUrl: 'https://your-domain.com'
```

#### 3.3 编译运行
1. 点击"编译"按钮
2. 在模拟器中查看效果

#### 3.4 测试功能
1. 在首页输入公司名称（如：腾讯控股有限公司）
2. 点击"开始分析"
3. 查看进度页面
4. 等待分析完成
5. 查看报告页面

## ❓ 常见问题

### Q1: 后端启动失败
**A**:
- 检查 Python 版本是否为 3.9+
- 检查是否安装了依赖：`pip install -r requirements.txt`
- 检查端口 8000 是否被占用

### Q2: 小程序无法连接后端
**A**:
- 检查后端服务是否正在运行
- 检查 `app.js` 中的 `apiBaseUrl` 是否正确
- 尝试在浏览器访问 http://localhost:8000

### Q3: Agent 初始化失败
**A**:
- 检查 `agent_wrapper.py` 中的路径是否正确
- 检查是否有权限访问 `src/agents/` 目录
- 查看错误日志
